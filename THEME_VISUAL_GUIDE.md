## Component Updates by Theme

### All Themes Update

When you click the Theme Changer button, the following components update instantly:

1. **Main Window Background** - Changes to theme's main background color
2. **Navigation Sidebar** - Changes to theme's darker background color
3. **Content Panels** - Update to theme's main background
4. **Card Panels** - Update to theme's card background
5. **Data Grids** - Update background, borders, and text colors
6. **Text Labels** - Update text color for visibility
7. **Input Fields** - Update background and text colors
8. **Buttons** - Update text color (accent buttons keep their colors)
9. **Status Badges** - Repaint with current theme colors

### Preserved Elements

These elements maintain their identity across themes:
- âœ… **Green badges** - For success/unlocked status
- âš ï¸ **Amber badges** - For warnings
- ğŸ”´ **Red button** - Emergency Flatten button
- ğŸ¨ **Blue theme button** - Theme changer itself

## Accessibility

### Contrast Ratios

All themes maintain WCAG 2.1 AA compliance for text contrast:

- **Blue Theme**: ~10:1 contrast ratio (White on Dark Blue)
- **Black Theme**: ~21:1 contrast ratio (White on Black)
- **White Theme**: ~16:1 contrast ratio (Dark on White)

### Visual Considerations

- **Blue Theme**: Balanced, professional
- **Black Theme**: Maximum contrast, best for OLED displays
- **White Theme**: Best for bright environments, reduces strain in daylight
